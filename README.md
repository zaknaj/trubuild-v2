# Trubuild v2

⚠️ don't use yarn, npm, or pnpm. use bun.

## Installation

`bun add` → `bun dev`

⚠️ make sure you have postgres running locally

add the following env vars

```
DATABASE_URL="postgresql://[USERNAME]:[PASSWORD]@localhost:5432/[DATABASE_NAME]"
BETTER_AUTH_SECRET=some random long key
BETTER_AUTH_URL=http://localhost:3000
```
